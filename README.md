## The Four Leverage Points

1. **Content** — 3–5× output with AI-augmented production workflows
2. **Attribution** — 20–35% ROAS improvement by fixing misattributed spend
3. **Segmentation** — 25–40% campaign efficiency gain via ML audiences
4. **Forecasting** — ±15% accuracy improvement with Prophet + bridge models

## Quick Start

1. Run the stack audit in `framework/05_stack.md`
2. Pick your highest-ROI lever based on your biggest constraint
3. Follow the 30-day pilot structure in the relevant module
4. Scale what works
